Git

Práctica con Git y GitHub

Pasos para clonar un proyecto y agregar modificaciones

  1. git clone (URI)
  2. git cd (carpeta)
  3. git branch (nombre de nueva rama)
  4. git checkout (nombre de la nueva rama)
  5. Realizar las modificaciones
  6. git status
  7. git add (archivo, carpetas, etc.)
  8. git status
  9. git commit -m "Mensaje"
  10. git push -u origin (nombre de la rama)

Pasos para subir un proyecto a GitHub

  1. Crear repositorio en GitHub
  2. cd (espacio de trabajo de Eclipse)
  3. cd (nombre carpeta)
  4. git init
  5. git add (archivo)
  6. git config --global user.name "Nombre y Apellido"
  7. git config --global user.email "email@gmail.com"
  8. git commit -m "Mensaje"
  9. git remote add origin (URI)
  10. git push -u origin (nombre de la rama)
